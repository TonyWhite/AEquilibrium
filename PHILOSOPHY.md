# Philosophy

This is not a toolkit. This is an asset.

It is created to be:
- Developer friendly
- User friendly
- Highly compatible

## Policy

- Vanilla HTML.
- Vanilla CSS.
- Vanilla JavaScript.
- No preprocessors.
- No frameworks.
- No external packages.
- No external references.
- No Artificial Intelligence.

This allows correct operation on all web platforms, even those based on frameworks.
